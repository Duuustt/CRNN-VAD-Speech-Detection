# Robust-VAD: 基于 CRNN 与复数域特征的稳健语音活动检测系统

## 1. 项目简介

本项目是一个端到端的语音活动检测（Voice Activity Detection, VAD）解决方案。系统能够在复杂背景噪声环境下，通过深度学习模型自动识别音频信号中的语音区间与非语音区间。

项目涵盖了从原始数据加噪增强、时频特征提取、模型构建到训练评估的全流程工程实践。

## 2. 核心技术特性

### 2.1 自动化数据增强 (Data Augmentation)

* **动态加噪**：支持根据指定的信噪比（SNR）将语音信号与随机噪声进行合成。
* **VAD 标签对齐**：自动根据原始标注生成帧级别的 0/1 标签，并处理静音阈值。
* **溢出保护**：采用信号限幅技术（Clipping）防止加噪过程中出现数值溢出。

### 2.2 复数域时频特征提取

* **STFT 双通道设计**：不同于传统的幅度谱，本项目提取短时傅里叶变换（STFT）的实部（Real）与虚部（Imag）并堆叠为双通道输入。
* **相位信息保留**：通过保留复数特征，模型能够获得更丰富的相位信息，提升在噪声环境下的鲁棒性。

### 2.3 CRNN 模型架构

* **空间特征提取**：利用三层 2D 卷积网络（CNN）捕捉频谱图在频率维度上的局部模式。
* **时序依赖建模**：集成门控循环单元（GRU）捕捉语音信号在时间跨度上的连续性特征。
* **非对称损失函数**：引入 **BCEFocalLoss**，通过调节 $\gamma$ 和 $\alpha$ 参数解决语音/非语音样本不均衡问题，聚焦于难分类的边缘样本。


## 4. 技术规格 (Technical Specifications)

* **采样率支持**：默认 16kHz。
* **分析窗长**：512 样本 (32ms)，步长 256 样本 (16ms)。
* **特征维度**：257 维频率分量。
* **核心超参数**：
* Optimizer: Adam (LR=0.0001)。
* Batch Size: 32。
* Loss: BCEFocalLoss (gamma=2, alpha=0.6)。



## 5. 如何复现

1. **准备数据**：将原始语音放入指定目录，并准备好 NOISE92 等噪声库。
2. **生成训练集**：修改 `split_wav.py` 中的路径并运行，生成 3 秒分段的加噪音频。
3. **启动训练**：运行 `python trainModel.py`。脚本会自动进行模型训练、验证以及保存最优权重。

## 6. 项目目前处于实验开发阶段
如果要完善它，需要实现以下逻辑：
* **加载模型**：使用 torch.load("VADModel.pth")。
* **音频切帧**：将待测音频按照训练时的 frame_size 和 hope_size 进行分帧。
* **特征变换**：对每一帧进行 STFT 变换，转化为模型需要的 2 通道张量。
* **后处理**：将模型输出的概率值（0~1 之间）通过一个阈值（如 0.5）转换为 0 或 1。
